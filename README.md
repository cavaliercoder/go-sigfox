# go-sigfox

A Go package for interfacing with the SigFox IoT network.

```
{
  "device": "{device}",
  "time": {time},
  "duplicate": {duplicate},
  "snr": {snr},
  "station": "{station}",
  "data": "{data}",
  "avgSnr": {avgSnr},
  "lat": {lat},
  "lng": {lng},
  "rssi": {rssi},
  "seqNumber": {seqNumber}
}
```
